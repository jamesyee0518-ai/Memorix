# Memorix Landing Page

This folder contains two versions of the bilingual landing page.

## Static preview

Open `preview.html` directly in a browser. It does not require package installation.

## Next.js source

The Next.js implementation lives in `app/page.tsx` with Tailwind CSS configuration.

When network/package installation is available:

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

The page defaults to Chinese and supports English through the language switcher.
